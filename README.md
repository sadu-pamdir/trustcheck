# TrustCheck

Offline trust score for websites. Coming to the Chrome Web Store.

TrustCheck shows an instant trust badge with a score from 0 to 100 on every website, so you can spot warning signs before you enter an email address, a password or payment details. It runs fully offline, sends nothing, and stores nothing remotely.

## What it does

- Shows a floating trust badge with a 0 to 100 score on every page.
- Scores the domain locally against an established brand list using the official Public Suffix List, plus a local pattern check on the URL (raw IPs, throwaway TLDs, phishing keywords) and a local scan for contact info and social links.
- Runs entirely in your browser, with no network calls, no accounts, no tracking, and no server of ours. Results are cached locally for 24 hours only.

TrustCheck is an early warning signal, not a guarantee. A high score does not mean a site is guaranteed safe, and a low score can also hit a harmless small site.

## Privacy

TrustCheck runs fully offline and sends no data. See [PRIVACY.md](PRIVACY.md) for the full privacy policy.
